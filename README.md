# AI 知识库（公开版）

## 简介

这是个人的 AI 对话知识库公开版本，存放通用技术、无隐私、无公司业务的会话记录。

## 目录结构

```
knowledge-public/
├── archive/          # 所有会话文件夹 全部平铺
├── template/         # 会话模板文件
├── assets-global/    # 全局公共资源
├── scripts/          # 自动化脚本
├── .gitignore
└── README.md
```

## 使用流程

1. 新建会话文件夹 → 按 `YYYY-MM-DD_序号_会话标题` 命名
2. 里面创建 `conversation.md` / `images/` / `files/` / `summary.md`
3. 粘贴对话内容、截图放 images、附件放 files
4. 提交到 GitHub

## 命名规范

- 文件夹：`YYYY-MM-DD_序号_简短标题`
- 示例：`2026-05-25_001_Java微服务线程池调优`

## 注意事项

- 本库存放公开内容，敏感信息请放 knowledge-private
- 会话全部平铺在 archive 目录，前期不做分类
